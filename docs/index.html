About Soft Meow CSS

What we include

One of the main features that I wanted to make was for the layout. This is because it is one of the things I struggle with the most while creating web pages. With flexbox, I had a difficult time with positioning the children or descendants of elements while banging my head on why an element stayed in the same position no matter what I tried. One thing that helped me with this was adding borders to every thing. I made all div elements into flex containers in order to make sure that child elements position the way I wanted on the layout.

I added different template font sizes in order to quickly choose a size that would looks presentable.

The last component I worked on was buttons. I love round buttons and one issue I used to have was having a rectangle appear after clicking on the button. These buttons have removed the rectangle and added a more molded outline after clicking.

Challenges

One of the most challenging aspects of this challenge was making the page more responsive to view size changes. I am still struggling with this aspect and will continue to figure it out.

The second challenge was the positioning of content after adding a fixed container, such as the navigation bubbles to the left.

Layout

FlexBox

Automatically uses flexbox for all the div elements. All descendants will be laid vertically as default.

To change the direction of the children inside an element, add 'row' to the element's class.

To modify the position, size, and alignment, use the flexbox properties.

This flexbox guide would be helpful understanding the different properties.

Borders

Layout helper borders will not appear on default.

To do so, add in the body element's class attribute 'addHelperBorders'.

These are meant to be used to help the creater visualize the layout for easier manipulation of flexbox.

Fonts

Default Font Familys

  • Our heading uses Source Sans Pro
  • All other content uses Open Sans

Font Sizes

Change the size of the fonts by adding these class selectors (xlg, lg, md, sm, xs, xxs). Headings can use any tags from (h1-h6). Will have padding and margin.
