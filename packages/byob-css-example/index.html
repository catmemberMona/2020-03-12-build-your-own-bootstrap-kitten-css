tech used

Layout

FlexBox

Automatically uses flexbox for the entire layout. All tags/elements will be laid vertically as default.

To change the direction of the children inside an element, add 'row' to the element's class.

To modify the position, size, and alignment, use the flexbox properties.

This flexbox guide would be helpful understanding the different properties.

Borders

Layout borders appear on default. They are used to better visualize the positions of the content.

Add in the body element's class attribute '.resetBorder' to remove layout borders

Fonts

Default Font Familys

  • Our heading uses Source Sans Pro
  • All other content uses Open Sans

Font Sizes

Change the size of the fonts by adding these class selectors (xlg, lg, md, sm, xs, xxs). Can use tags from (h1-h6). Will have padding and margin.
